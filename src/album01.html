Strange Sensation

2000

In 2000, two friends, a certain Irony and Fly, decided that it would be better if they put their creations into one album, the name of which has become a frequent expression in their compositions.
At first it was a game, then a hobby, flowed into a hobby and finally it became a vocation. This is how the First Project from D3 and the D3 Misty Project itself appeared. It should be noted that D3 and D3 Misty Project are not the same thing.
Strange Sensation was not originally created as an album. As an album, these compositions came together on October 10, 2000. This is not an easy date. 10.10.2000 how round the date is, the more mysterious it is.
On this day, the first disc of this still misty project was released.
